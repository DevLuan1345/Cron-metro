## ⏱️ Cronômetro Web

Este projeto é uma aplicação web de cronômetro digital, desenvolvida com o objetivo de praticar conceitos fundamentais de HTML, CSS e JavaScript, criando uma ferramenta interativa para medição de tempo em tempo real.

O cronômetro permite iniciar, pausar, resetar e registrar marcações (laps), oferecendo uma experiência simples e funcional diretamente no navegador, sem necessidade de instalação. Ferramentas desse tipo são amplamente utilizadas para controle de tempo em atividades como estudos, treinos e produtividade .

A interface apresenta um design moderno com destaque para o visor circular do tempo, botões intuitivos com ícones e uma organização limpa, garantindo boa usabilidade em diferentes dispositivos.

## 🚀 Funcionalidades

- ▶️ Iniciar e pausar o cronômetro
- 🔄 Resetar o tempo (zerar)
- 🚩 Registrar marcações de tempo (laps)
- ⏱️ Contagem precisa em horas, minutos, segundos e milissegundos
- 📱 Interface responsiva e adaptável

## 🎨 Interface

- Design moderno com destaque visual para o cronômetro
- Display central em formato circular
- Botões com ícones intuitivos (play, reset, marcação)
- Layout limpo e organizado

## 🛠️ Tecnologias utilizadas

- HTML5 🟧
- CSS3 (Flexbox e estilização moderna) 🟦
- JavaScript 🟨
- Font Awesome (ícones) 🔮

## 📱 Objetivo

O principal objetivo deste projeto é praticar manipulação do DOM e lógica de tempo, além de desenvolver uma aplicação interativa que simula um cronômetro real utilizado no dia a dia.
